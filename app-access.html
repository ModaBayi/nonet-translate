Review guidance

App Access

Last updated: May 17, 2026

This page exists to support app review preparation for NoNet Translate.

Current publishing intent

The recommended baseline release path is a build that can be reviewed without account creation or sign-in.

  • If the app does not require login, state this directly in Google Play's App access section.
  • If a future build requires login, provide reviewer credentials only in Play Console.
  • Do not publish active reviewer credentials on a public page.

Operational note

This public page should be treated as guidance only. The authoritative review access details should be entered in Play Console for the exact build under review.
