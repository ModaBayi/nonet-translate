Why this repo exists

Public docs without publishing source code

Google Play and similar stores often require a public privacy policy, support contact, and a stable app information page. This repository is designed to hold those public resources while the app source remains private.

For store review

Prepared for store listing work

The linked documents are structured so they can be reused when filling Play Console fields such as privacy policy, developer contact, app description support links, and review preparation notes.
