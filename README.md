# NoNet Translate

This repository hosts the public website and public compliance documents for the NoNet Translate mobile app.

It is intentionally separate from the private application source code.

## Public documents

- `index.html`: project landing page
- `privacy-policy.html`: privacy policy for Google Play and user reference
- `support.html`: support and contact page
- `app-access.html`: public review guidance page
- `play-console-prep.md`: working notes for Play Console setup

## GitHub Pages

When you are ready to publish the site, enable GitHub Pages for the `main` branch and the repository root in the repository settings.
